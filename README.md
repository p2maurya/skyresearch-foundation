# SkyResearch Foundation Website 🚀

A modern PHP + MySQL based internship application website with responsive UI, program section, and contact form with popup notifications.

---

## 🔥 Features

- Modern Axe-Selection Inspired UI
- Responsive Navbar & Hero Section
- Internship Programs Section
- Contact Form with Success/Error Popup
- MySQL Database Integration
- Clean Footer Design
- Fully Responsive Layout

---

## 🛠 Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: PHP
- Database: MySQL
- Hosting: Compatible with XAMPP / InfinityFree / 000webhost

---

## 📂 Project Structure
skyresearch/
│── index.php

│── contact_form.php
│── submit.php
│── db.php
│── css/
│── images/

---

## ⚙️ Installation (Local Setup)

1. Install XAMPP
2. Copy project into:
   C:\xampp\htdocs\
3. Create MySQL database named:
   skyresearch
4. Import provided SQL file
5. Start Apache & MySQL
6. Open:
   http://localhost:8081/skyresearch
